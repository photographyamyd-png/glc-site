## Track E — `glc-dna-extracted.css` `.eyebrow` vs `globals.css`

| Source | Spec |
|--------|------|
| [`app/globals.css`](../../app/globals.css) `.eyebrow` | Barlow Condensed, **13px**, **600** (rolled out from 700), `0.07em` |
| [`app/glc-dna-extracted.css`](../../app/glc-dna-extracted.css) `.eyebrow` (~58–63) | **`var(--font-body)`**, **11px**, **`font-weight: 600`** (rolled out from 800), `0.15em` — different DNA stack |

**Import graph:** Root layout loads **`globals.css`** only. DNA lane imports **`components/glc-dna/glc-dna-lane.css`** → **`glc-dna-extracted.scoped.css`**, generated by `npm run scope:glc-dna` from `app/glc-dna-extracted.css`. **No** route loads unscoped `glc-dna-extracted.css` and globals on the **same** node by default; collision risk is **class name reuse** if a node ever received both global and unscoped DNA sheets (unlikely in current wiring).

**Rollout decision:** Align DNA `.eyebrow` weight **800 → 600** for a lighter kicker in the sandbox/export lane (still **body** face at 11px — not identical to main site Barlow eyebrow). Regenerate scoped + intro export via **`npm run scope:glc-dna`** after editing the source file.
